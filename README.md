# 📊 Student Performance Analysis

## 📌 Project Overview
This project analyzes student lifestyle factors such as study hours, sleep, stress, and social activity to understand their impact on academic performance (GPA).

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Correlation analysis
- Data visualization using scatter plots, heatmaps, and regression plots

## 📈 Key Insights
- Study hours show a strong positive impact on GPA
- Stress levels negatively affect academic performance
- Adequate sleep contributes to better GPA
- Social activities have a slight negative influence

## 📌 Conclusion
The analysis highlights that balanced study habits, proper sleep, and stress management are key factors for better academic performance.
